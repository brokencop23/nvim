Neovim Cheatsheet

Tokyo Night — All keybindings from your config

Leader key: Space

Core Navigation

N Ctrl+h/j/k/lMove between windows
N Ctrl+Up/DownResize window height
N Ctrl+Left/RightResize window width
N Shift+lNext buffer
N Shift+hPrevious buffer
N SpacebdDelete buffer

Scrolling & Search

N Ctrl+dScroll down half page (centered)
N Ctrl+uScroll up half page (centered)
N n / NNext/prev search result (centered)
N EscClear search highlighting

Editing

N SpacewSave file
N Ctrl+sSave file
N SpaceqqQuit all
V < / >Indent left/right (stays in visual)
V J / KMove selected lines down/up
X SpacepPaste without replacing clipboard
NV SpaceyYank to system clipboard
N SpaceYYank line to system clipboard
N gccToggle comment (line)
V gcToggle comment (selection)

Telescope (Fuzzy Finder)

N SpaceffFind files
N SpacefgLive grep
N SpacefbFind buffers
N SpacefhHelp tags
N Ctrl+pFind git files
N SpacepwsGrep word under cursor
N SpacepWsGrep WORD under cursor
N SpacepsGrep string (prompted)
N SpacedsDocument symbols
N SpacewsWorkspace symbols
N SpaceftFind TODOs
I Ctrl+j/kNavigate results up/down

LSP (Code Intelligence)

N gdGo to definition
N gDGo to declaration
N giGo to implementation
N grReferences (Trouble)
N gRReferences (quickfix)
N SpacehHover documentation
N SpacecaCode action
N SpacernRename symbol
N SpacecfFormat buffer
N SpaceceLine diagnostics (float)
N [dPrevious diagnostic
N ]dNext diagnostic
N SpaceuhToggle inlay hints

Completion (nvim-cmp)

I Ctrl+SpaceTrigger completion
I Ctrl+nNext item
I Ctrl+pPrevious item
I Ctrl+yConfirm selection
I EnterConfirm (if selected)
C /Buffer search completion
C :Command + path completion

Treesitter (Selection & Motion)

INCREMENTAL SELECTION
N EnterStart selection
V EnterExpand to next node
V BackspaceShrink to previous node
V Shift+EnterExpand to scope
TEXT OBJECTS
af / ifAround/inside function
ac / icAround/inside class
aP / iPAround/inside parameter
MOVE & SWAP
N ]m / [mNext/prev function
N ]c / [cNext/prev class
N SpacespSwap parameter forward
N SpacesPSwap parameter backward

Trouble (Diagnostics)

N SpacexxToggle Trouble

N SpacexdDocument diagnostics
N SpacexlLocation list
N SpacexqQuickfix list
